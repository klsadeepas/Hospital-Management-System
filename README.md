# Hospital-Management-System
A Hospital Management System built using React (frontend-focused) is a web-based application that simulates hospital operations entirely on the client side.

# How can I run this code?
The only requirement is having Node.js & npm installed

# What technologies are used for this project?
This project is built with:
- Vite
- TypeScript
- React
